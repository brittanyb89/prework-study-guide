Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section elements are semantic elements that define a section in a document.
  • The meta element defines metadata such as character set, description, keywords, author, and viewport.
  • The link element connects an external resource to the HTML document.
  • HTML is the foundation of the website.
  • The base element defines URL for webpage.

CSS

  • CSS is the appearance of a website.
  • The style element defines CSS styles associated with website.
  • Using CSS, I can add boarders, change colors and fonts, and make the site pretty.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • We use Git in our local development environment to enable version control.
  • Git is a software on your computer that tracks changes that you make.
  • In bootcamp, we will utilize GitHub, GitBash, and GitLab and know the difference.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • JavaScript brings the functionality to the website.
  • JavaScript handles the logics and allow users to interact with website.
  • The script element embeds client-side scripts in the HTML document.
  • This element can either contain scripting statements or it points to an external source attribute.
  • common uses of JavaScript: manaipulate images, form validation, and dynamic changes of content.
